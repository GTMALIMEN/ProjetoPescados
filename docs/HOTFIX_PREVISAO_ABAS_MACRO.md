# Hotfix — Abas macro na Análise Previsão de Mercado

## Objetivo

Dividir a aba **Análise Previsão de Mercado** em abas internas:

```text
Cotação Dólar
IPCA
IPCA Alimentar
Previsão Mercado
```

## Organização

### Cotação Dólar

Mostra indicadores relacionados a:

```text
dólar
dolar
usd
cambio
câmbio
ptax
```

### IPCA

Mostra indicadores IPCA gerais, excluindo alimentação.

### IPCA Alimentar

Mostra indicadores relacionados a IPCA e alimentação.

### Previsão Mercado

A partir de **Proteínas e grãos consolidados**, o conteúdo continua dentro desta aba:

```text
Proteínas e grãos consolidados
Bases de mercado
CEPEA
CEAGESP
Base de Compra
Prévia Vendedores
Índices
Exportação
```

app.py
