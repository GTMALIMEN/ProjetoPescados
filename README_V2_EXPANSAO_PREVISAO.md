# Radar Pescados IA — Versão 2.0

## Visão geral

A versão 2.0 organiza o projeto em duas frentes:

```text
🌎 Análise de Expansão
📈 Análise Previsão de Mercado
```

A aba **Análise de Expansão** complementa a antiga aba de Potencial Regional.

A aba **Análise Previsão de Mercado** é uma nova área para preços, custos, proteínas, grãos, compra, CEAGESP e previsão.

## Etapa 22
